# 🐢 Turtle Tracking Project

A multi-language demo project inspired by turtles and marine conservation systems.

## Features
- Turtle movement simulation (Python)
- Habitat tracker (JavaScript)
- Web interface (HTML/CSS)
- Simple API server (Node.js)
- Migration engine (C++)
- Turtle model (Java)
- Swim distance calculator (Go)
- Diagnostics tool (Rust)
- Configuration files (JSON & YAML)

Protect marine life through technology.
